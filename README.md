# To-Do List App
Thi is a simple To-Do App which i made using HTML , CSS and JS . In this to-do app you can store the things you want to do according to the category . It helps to organize the tasks 
more appropriatley . It is also responsive which helps to make it easier to navigate through both mobile devices and laptops. Moreover you can navigate through dark mode and light mode both
in this web app.
